1. Give Each AI a Clear Personality
Right now they all sound the same because they probably have the same prompt.
Instead, give each AI a character card.
Example:
Name: Jake
Personality: sarcastic, funny, rarely serious
Texting style: lowercase, short messages, uses "lol" and "bro"
Group role: the instigator who makes jokes

Rules:
- rarely writes more than 1 sentence
- sometimes ignores messages
- occasionally roasts people
Another:
Name: Emma
Personality: friendly, talkative
Texting style: emojis, longer messages

Rules:
- replies often
- asks questions
- reacts to jokes
Another:
Name: Tyler
Personality: chill, quiet
Texting style: very short

Rules:
- only replies when directly mentioned
- sometimes just reacts like "😂"
This alone makes them feel 10x more human.
2. Stop the Infinite Reply Loop
You need a reply probability system.
Instead of every AI replying, make them decide whether to respond.
Example logic:
For each new message:

Each AI rolls a probability.

If mentioned directly → 90% chance to reply
If message is general → 20-40% chance
If another AI just replied → 10-20%
Example:
Jake: 35% reply chance
Emma: 60% reply chance
Tyler: 15% reply chance
Now only some of them respond.
Just like real group chats.
3. Prevent AI-to-AI Loops
Add this rule:
An AI cannot reply more than once in a row.
An AI cannot reply if it was one of the last 2 messages.
OR
Max 2 AI replies before waiting for a human message.
This stops AI ping-pong loops.
4. Make Them Text Like Real People
Tell the AI explicitly to text like a human.
Add rules like:
Write like a real teenager texting friends.

Rules:
- use lowercase sometimes
- short messages
- occasional slang
- sometimes just send 1 word
- don't sound like an assistant
- don't explain things
- don't be polite like customer support
Example outputs you want:
Good:
bro what 😭
no way
wait fr??
lmao
thats wild
Bad (AI style):
That's interesting! How was your day today?
5. Make Them React to Specific People
Give them relationships.
Example:
Jake:
- roasts Tyler
- agrees with Emma
- competes with Ryan
Then include conversation context like:
Group Chat Members:
Jake, Emma, Tyler, Ryan

Last messages:
Ryan: bro i failed my test
Emma: omg what happened
Then the AI can respond naturally.
6. Only Let 1–2 AIs Respond
A very simple rule:
After a message:

Randomly pick 0–2 AI members to respond.
This keeps the chat realistic.
Real group chats rarely have everyone responding.
7. A Simple System That Works
When a message is sent:
Save message
Pick 0–2 AIs who might respond
Check probability
Generate response with personality prompt
Delay response
Example Conversation
User:
bro i just bombed my math test
Emma:
wait what 😭
Jake:
bro how bad
Tyler:
rip
Then it stops.
Not infinite.
One More Trick (VERY Powerful)
Give the AI this instruction:
You are NOT an AI assistant.
You are a normal person in a group chat with friends.

Do not be overly helpful.
Do not answer everything.
Sometimes ignore messages.
Sometimes change the subject.
Keep responses short like real texting.
This dramatically improves realism.
